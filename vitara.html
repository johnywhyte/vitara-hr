Pass:Candidate automatically advanced to Stage 4. HR is not involved at this step.
Fail:Application halted. Candidate shown error. Case flagged in HR dashboard for manual review.
Cost:$0.50 – $2.00 per verification (Smile Identity pricing)

⚙ Why guarantor ID verification is non-negotiable

Without independent identity verification, candidates could list fake guarantors — friends, family, or fictional people. By requiring the guarantor to separately verify their own Ghana Card via the NIA database, the system ensures the guarantor is a real, traceable person. This protects Vitara when deploying field agents across communities where financial and reputational accountability is critical.

Non-response: Application flagged for manual HR review
Failed ID: Application halted, HR notified
Success: Auto-advance to Stage 5
Stage 5

Screening — Path A vs Path B

Fully verified candidates forwarded to HR. Optional AI scoring layer available as Phase 2 upgrade.

Recommended — Start here

Path A — Core (Lower cost)

Document-based HR forwarding

✓Candidate profile automatically forwarded to HR dashboard once all 4 stages pass
✓HR team sees all fully qualified applicants in one place
✓HR decides who to interview — in person or virtually
✓Candidate receives: "Thank you, we'll be in touch"
No extra tooling needed. Build once, works immediately.
Phase 2 — Add when ready

Path B — Enhanced (+ AI layer)

AI-assisted candidate scoring

→AI model scores each verified applicant against pre-set criteria (Shea experience, location, qualifications)
✗Below 60%: automatic rejection email within 3 days
✓Above 60%: profile forwarded to HR for human review & interview
Setup cost: $500–$2,000+ plus API costs per batch. Build on top of Path A — no rebuild needed.
